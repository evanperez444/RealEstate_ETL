# RealEstate_ETL
----------------
Project Goals: Database Management, Data Transformation, Automation, Analytics

* This is a personal project that aims to:
  * extract Real Estate price data using RapidAPI.
  * transform the data by aggregating statistics that describe the relationship between attributes and combine the data with other sources such as city population or crime rate.
  * Load the transformed data into an SQL database and design a schema for the database.
  * Additional: Include data visualizations and real time dashboards
